# 💕 Álbum Digital Romántico — Premium Edition

Álbum interactivo de fotos con estética atardecer romántico, cuerdas SVG Bézier, carrusel infinito, y experiencia premium. 100% HTML + CSS + JS puro — cero frameworks.

## 📁 Estructura
```
index.html              → Página principal (parallax + capas)
css/styles.css          → Estilos premium (sunset, glassmorphism, SVG)
js/data.js              → Datos y configuración — solo edita este archivo
js/app.js               → Motor: cuerdas SVG, carrusel, modal, typewriter, audio
assets/images/          → Tus fotos (01.jpg a 08.jpg)
assets/music/           → Tus canciones (01.mp3 a 08.mp3)
```

## ✨ Características premium
- Fondo atardecer con parallax de 4 capas
- Brillos dorados flotantes
- Corazones SVG semi-transparentes
- Cuerdas SVG con curvas Bézier (caída natural)
- Luces cálidas alineadas dinámicamente a la curva
- Carrusel infinito con auto-scroll y drag
- Pinzas de madera realistas
- Modal con controles de música y slider de volumen
- Typewriter inteligente (pausa en comas y puntos)
- Pantalla de acceso con glassmorphism y glow animado
- Grain texture sutil para profundidad visual

## 🚀 Personalizar
1. Edita `js/data.js` — fotos, textos, música, fecha clave
2. Coloca imágenes en `assets/images/` (01.jpg, 02.jpg...)
3. Coloca música en `assets/music/` (01.mp3, 02.mp3...)
4. Abre `index.html` en el navegador
